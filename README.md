# Traffic-Accident-Analysis
Traffic Accident Data Analysis using Python and Tableau. This project explores accident trends through data visualization and dashboard development to identify patterns in time, location, and contributing factors.


## Dataset
Source: Kaggle – Traffic Accident Dataset

The dataset contains information about traffic accidents, including variables such as date, time, location, accident severity, weather conditions, and other related attributes.

## Tools
- Python
- Pandas
- Matplotlib
- Tableau

## Objectives
- Explore accident trends
- Create visualizations
- Build an interactive Tableau dashboard

## key findings
Some of the insights obtained from the analysis include:
- Peak accident periods during the day.
- Trends in accident frequency over time.
- Distribution of accident severity.
- Relationships between accident conditions and outcomes.

•	 Accidents follow clear patterns and they’re not random.
•	 High-risk factors are : intersections, daytime driving, and common driver errors.
•	Some crashes are more dangerous, even if less frequent.
